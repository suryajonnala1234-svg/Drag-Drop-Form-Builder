# Drag and Drop Form Builder

A simple drag-and-drop form builder built using HTML, CSS, and JavaScript.

## Features
- Drag & drop form elements
- Editable headings and labels
- Text input, textarea, select, radio, checkbox
- Dynamic form generation
- Data stored using LocalStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open `index.html`
2. Build your form
3. Click Submit to generate final form
